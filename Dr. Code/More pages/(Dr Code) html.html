HTML


Hypertext Markup Language

What is HTML?

HTML is the structure of a website and it is always used when making online web pages. There are also frameworks or extensions for this language like Haml and more

HTML is used for what?

This tag programming langauge is used to build websites by adding heading, paragraphs, sections, organizations, arrangement, videos, pictures, lists, tables, and more!

Is HTML still used today?

It is still used today by thousands and more for creating websites types like E-commerce, portfolios, blogs, and more.

Should I learn HTML in 2021?

Learning HTML is a good thing because it is a very famous programming language and you can use more programming and scripting languages with it like CSS, JavaScript, PHP, Java, and more.

Is HTML easy?

HTML is easy to understand and implement with. Although, there are many elements in HTML, it is still easy to read and organize.

Where can I upgrade my skills in HTML?

There are many pages on the internet that teaches programming language (including HTML) like W3Schools, FreeCodeCamp.org, and more.

HTML Syntax Examples

Headings & Paragraphs

                        
                            <h1>Biggest Heading</h1>
                            <p>Paragraph</p>
                            <h6>Smallest Heading</h6>
                            <p style="color: pink;">Colored Paragraph</p>

Division

                        
                            <div class="main">
                              <a href="www.google.com">This is an anchor</a>
                              <strong>This text is bold</strong>
                            </div>

Unordered List

                        
                            <ul>
                              <img src="download.jpg" type="image/jpg">This is an anchor</img>
                              <li>A list item. Currently the child of the ul element</li>
                              <em id="more-info">This text is italic</em>
                            </ul>
